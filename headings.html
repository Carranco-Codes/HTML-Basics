Headers

Description

HTML headings are titles or substitles that you want to display on a webpage.

Guidelines for using headers

  • each heading tag should have an opening and closing tag.
  • each heading should be used consistently throughout your site.
  • you should use headings as titles of a specific section in your site.
  • each HTML page should include only one **< h1 >** tag.
